# Features

  • Render static pages fastly (Use the pug template engine)
  • Smoothly update (Use the Hexo data files feature)
  • Comments support
  • Share support
  • Search support
  • Analytics support
  • Animation support
  • Catalog support
  • MathJax support
  • i18n support
  • Compatible with IE10+
  • ...
